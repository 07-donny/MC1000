Hoe vul ik de database met data? Goeie vraag!
1. Log in met de admin account.
2. Ga naar het CMS
3. Klik op Hangfire
4. Klik boven aan het scherm op Herhalende Taken
5. Selecteer XML Data Ophalen.
6. Klik op de knop Nu Uitvoeren
7. Even wachten..
8. Data vers vanuit de API is nu zichtbaar op de website.
